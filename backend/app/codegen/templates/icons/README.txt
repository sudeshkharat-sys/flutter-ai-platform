Place your app icon here as:

    ic_launcher.png

Requirements:
  - Format : PNG with transparency (recommended) or JPEG
  - Minimum size : 512 x 512 pixels (larger is better — it will be downscaled)

The generator will automatically resize it to all required Android mipmap densities:
  - mipmap-mdpi      48 x 48 px
  - mipmap-hdpi      72 x 72 px
  - mipmap-xhdpi     96 x 96 px
  - mipmap-xxhdpi   144 x 144 px
  - mipmap-xxxhdpi  192 x 192 px

If no icon is placed here, the app will use the default Android launcher icon.
